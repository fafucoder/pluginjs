# Feature

[![npm package](https://img.shields.io/npm/v/@pluginjs/feature.svg)](https://www.npmjs.com/package/@pluginjs/feature)

`feature` is a utility JavaScript library for special helper.

---

## API

- prefixedProperty
- transitionProperty
- transformProperty
- animationProperty
- transitionEndEvent
- animationEndEvent
- transform3D
- transform
- transition
- canSticky
- isSupportedSvg
- touch
- pointer
- pointerEvent
