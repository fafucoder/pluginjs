# I18n

[![npm package](https://img.shields.io/npm/v/@pluginjs/i18n.svg)](https://www.npmjs.com/package/@pluginjs/i18n)

`i18n` is a utility JavaScript library for globalization.

---

## API

- hasTranslation
- addTranslation
- getTranslation
- instance
- setTranslations
