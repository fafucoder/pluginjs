# Keyboard

[![npm package](https://img.shields.io/npm/v/@pluginjs/keyboard.svg)](https://www.npmjs.com/package/@pluginjs/keyboard)

`keyboard` is a utility JavaScript library for keyboard event.

---

## API

- registerEvent
- handler
- Each
- on
- off
- dispatch
- parseKeys
- processKey
- processModifiers
- distribute
- getKeyName
- getKeyCode
- up
- down
