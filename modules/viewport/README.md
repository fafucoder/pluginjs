# Viewport

[![npm package](https://img.shields.io/npm/v/@pluginjs/viewport.svg)](https://www.npmjs.com/package/@pluginjs/viewport)

`viewport` is a utility JavaScript library to control viewport.

---

## Usage

```Javascript
new Viewport(el, options)
```

## API

- on
- off
- eventMapper
- isVisible
