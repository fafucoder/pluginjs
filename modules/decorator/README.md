# Decorator

[![npm package](https://img.shields.io/npm/v/@pluginjs/decorator.svg)](https://www.npmjs.com/package/@pluginjs/decorator)

`decorator` is a class decorator library for extend component class.

---

## API

- register
- eventable
- stateable
- styleable
- themeable
- translateable
