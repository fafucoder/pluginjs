# MatchSorter

[![npm package](https://img.shields.io/npm/v/@pluginjs/match-sorter.svg)](https://www.npmjs.com/package/@pluginjs/match-sorter)

`match-sorter` is a utility JavaScript library.

---

## API

- matchSorter
- getHighestRanking
- getMatchRanking
- getAcronym
- stringsByCharOrder
- sortRankedItems
- prepareValueForComparison
- getItemValue
- getAllValuesToRank
